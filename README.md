# Snake

A browser-based Snake game with mobile support. Built with vanilla JS and HTML5 Canvas.

## Features

- Keyboard (arrow keys / WASD) and touch controls (swipe + on-screen d-pad)
- Pause with spacebar, Escape, or tap
- Local high score tracking
- Particle effects, screen shake, pulsing food
- Responsive sizing for mobile and desktop

## Run locally

```bash
npm install
npm start
```

Open http://localhost:3000

## Deploy to Railway

```bash
railway init
railway up
railway domain
```
