# Snake

A browser-based Snake game with mobile support. Built with vanilla JS and HTML5 Canvas. The goal was to make a simple Snake game that works on mobile and desktop.

## Features

- Keyboard (arrow keys / WASD) and touch controls (swipe + on-screen d-pad)
- Pause with spacebar, Escape, or tap
- Local high score tracking
- Particle effects, screen shake, pulsing food
- Responsive sizing for mobile and desktop

## Run locally

```bash
npm install
npm start
```

Open <http://localhost:3000>
